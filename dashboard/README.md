# Power BI Dashboard

## Overview
This folder contains the Power BI dashboard used to analyze workforce budgets, project spending, and department performance. The dashboard helps identify departments and projects that may exceed their allocated budget.

## Dashboard Features

### Employee Profile
Displays employee information including:
- Employee ID
- Name
- Job Title
- Department
- Total Compensation

### Budget Indicators
Key KPI visuals show:
- Capital available
- Total project budget

These indicators help evaluate financial capacity and spending levels.

### Department Budget Analysis
Compares:
- Project costs
- Salary costs
- Total department budget
- 2-year budget allocation

This helps identify departments at risk of exceeding budget.

### Project Budget Distribution
Visualizations display:
- Budget allocation across projects
- Project budgets by department
- High-cost projects

### Interactive Filters
Users can filter the dashboard using:
- Department name
- Project status
- Employee ID

These filters allow detailed analysis of workforce and financial performance.
